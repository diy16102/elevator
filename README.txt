Elevator simulation
Instruction:
Simply click java jar in a system correctly installed java runttime enviroment. 

Elevator moving:
Total five elevator.
Select desired floor in up or down button categories. 
The closest 'elevator' will move towards to desired floor.
Bottom button for emergency stop and make elevator out-of-order. Re-click to release elevator.
Button color indicates status of elevator. I.E blue shows while elevator stops will seek for passanger in the current floor.
